
git init . 	-> Cria um repo git na pasta atual
git status 	-> Mostra o status do trabalho
git add -> adiciona arquivo para ser commitado
git commit -m -> faz commit do que estava pra ser commitado  
git lop -> mostra todo histórico de commit


## FLUXO DE TRABALHO

1. Criar o diretorio do projeto
2. iniciar o git( Git init . dentro do diretorio)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m "mensagem estranha"
8. git status
9. volta para o passo 3.
